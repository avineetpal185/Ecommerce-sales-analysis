# E-commerce Sales Data Analysis

## Dataset
This project analyzes an E-commerce dataset taken from Kaggle.

## Tools Used
- Python
- Pandas
- Plotly
- Jupyter Notebook

## Analysis Performed
1. Profit by Category
2. Monthly Sales Trend
3. Sales vs Profit Relationship
4. Loss Making Products
5. Profit Margin Analysis

## Key Insights
- Technology category has the highest profit margin.
- Tables sub-category generates high sales but negative profit.
- Different categories contribute differently to profit.
